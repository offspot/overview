# Wikipedia

## **Purpose & Audience**
Thw pack provides **offline access to the entire Wikipedia, Wiktionary, and includes also some educational science simulations and technical resources**. It serves educators, students, researchers and anyone who want to have access to Wikipedia without having the need to be connected to the Internet.

## **Pack Contents**
| ZIM Files and apps              | Description                                      |
|------------------------|--------------------------------------------------|
| wikipedia              | The free online encyclopedia, created and maintained by a community of volunteers worldwide.    |
| wiktionary             | A multilingual, web-based project to create a free content dictionary of terms (including words, phrases, proverbs, linguistic reconstructions, etc.)  |
| phet-simulations       | Interactive simulations helping students engage in science and mathematics.   |
| raspberry-pi-qna       | Stack Exchange Q&A for users and developers of Raspberry Pi hardware or software  |
